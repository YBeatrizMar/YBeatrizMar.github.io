Humanidades digitales prueba

Betticita´s website

El autor propone un marco estructural para pensar y construir mundos virtuales que integren preocupaciones típicas de humanidades:

Documentación

• Documentación: cómo se captura y representa información cultural o textual en entornos digitales.

Anotación

• Anotación: marcar y enlazar elementos semánticos dentro de un espacio virtual (conceptualmente parecido a etiquetas o metadatos).

Interpretación

• Interpretación: el uso del espacio virtual como herramienta interpretativa para experiencias o textos.

Argumentación

• Argumentación: cómo los mundos virtuales pueden servir como espacios para presentar hipótesis, narrativas o argumentos científicos/humanísticos.
